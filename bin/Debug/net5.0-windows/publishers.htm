Publishers Input Form

Available options for managing Publishers database table:

Add New Record

Click the Add New button to add a record to the Publishers database. Type in the requested fields.Type in the Publisher Name is a required field. Click Save to save the new record; click Canecl to ignore the new record.

Edit Record

Click the Edit button to edit the displayed record. Make any needed changes. The Publisher Name is required. Click Save to save the changes; click Cancel to ignore the changes.

Delete Record

Click the Delete button to delete the displayed record.

Exit Program

Click the Done button to quit the application.
